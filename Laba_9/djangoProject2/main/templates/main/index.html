Финансовые переводы

На данном сайте есть 3 роли:
1) User - пользователь сайта, имеет доступ к таблице "операции" и "переводы",
может выполнить только операцию добавления записи

2) Teller - Операционист, занимается выпуском карт, личными данными клиентов.
Имеет доступ к таблицам "Участники", "Банк", "Карта", "Личные данные" и может
выполнять все операции с таблицами

3)Admin - администратор сайта, имеет доступ ко всем таблицам, может выполнять
все операции со всеми таблицами
